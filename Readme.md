### 自建节点转订阅
* 在线`Subscription Converter`平台;推荐下[https://sub.xeton.dev/](https://sub.xeton.dev/)

* 以自建Trojan节点为例:
```
	#每行一个;例: trojan://foobar@yjiang.cn:443?peer=#yjiang
	trojan://{password}@{host}:{port}?peer=#{alias}
```

[](./example.png)

### 注意事项:
* 需要勾选`输出为Node List`选项
* 更多选项中，需要根据情况是否勾选`跳过证书验证`
